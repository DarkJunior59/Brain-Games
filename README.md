To install "Brain-Games" you need:

1. Clone [repo](https://github.com/DarkJunior59/frontend-project-lvl1)
2. Run the following commands in the root directory:

 ```sh 
 $ make install
 ```

```sh
$ make publish
 ```

```sh
$ npm link
 ```

To start games, you need to run the following commands:

```sh 
$ brain-even
```
